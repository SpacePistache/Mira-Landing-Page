About the Project

Project Mira was inspired by the need for objective, non-verbal visual testing — especially for preverbal children or individuals with communication challenges. Rather than relying on patient feedback, Mira uses direct eye movement analysis to perform accurate visual acuity tests via the web.

Built as part of the Holberton School Portfolio Project. See full source on GitHub.
